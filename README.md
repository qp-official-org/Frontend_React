## How to run

```bash
yarn install
yarn start
```

## How to build

```bash
yarn build
```

## How to test

```bash
yarn test
```

## How to lint

```bash
yarn lint
```

## How to format

```bash
yarn format
```

## How to run lighthouse CI

```bash
npm install -g @lhci/cli@0.8.x
lhci autorun
```
